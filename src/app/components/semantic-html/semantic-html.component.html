Semantic HTML

In this section, we will learn about using semantic HTML elements to create meaningful and accessible web pages.

Semantic HTML refers to the use of HTML tags that provide meaning to the structure and content of a webpage. These tags include elements such as

<header>, <nav>, <main>, <section> and <footer>
etc. By using these tags, search engines and assistive technologies can understand the purpose and structure of the content on a webpage.

For example, instead of using a 

<div>
tag to create a header section, we can use the 
<header>
 tag. This indicates to search engines and assistive technologies that the content within this tag is a header section, rather than just a generic container.

It's also important to note that semantic HTML is not only beneficial for accessibility and SEO, but it also makes it easier for developers to understand and maintain the code.

Example:

          
  <header>
    <nav>
      <ul>
        <li><a href="#home">Home</a></li>
        <li><a href="#about">About</a></li>
        <li><a href="#services">Services</a></li>
        <li><a href="#contact">Contact</a></li>
      </ul>
    </nav>
  </header>
  <main>
    <section id="home">
      <h2>Home</h2>
      <p>Welcome to our website!</p>
    </section>
    <section id="about">
      <h2>About</h2>
      <p>Learn more about our company and our mission.</p>
    </section>
    <section id="services">
      <h2>Services</h2>
      <p>Discover the services we offer to our
      clients and how we can help you achieve your goals.</p>
      </section>
      <section id="contact">
        <h2>Contact</h2>
        <p>Get in touch with us to learn more or to schedule a consultation.</p>
      </section>
    </main>
    <aside>
      <h3>Related Links</h3>
      <ul>
        <li><a href="#">Accessibility Guidelines</a></li>
        <li><a href="#">SEO Best Practices</a></li>
      </ul>
    </aside>
    <footer>
      <p>Copyright Â©2022 My Website</p>
    </footer>

As you can see in the example, we are using semantic HTML tags such as 

<header>, <nav>, <main>, <section> and <footer>
to create a meaningful and structured webpage.
