Canvas

The HTML <canvas> element is used to draw graphics on a web page. It can be used to create animations, games, and other interactive content.

Canvas Example

In this example, we will create a simple canvas that displays a red rectangle.

          
  <canvas id="myCanvas" width="200" height="100"></canvas>
  <script>
    var canvas = document.getElementById("myCanvas");
    var ctx = canvas.getContext("2d");
    ctx.fillStyle = "red";
    ctx.fillRect(10, 10, 180, 80);
  </script>

In this example, we first create a canvas element with a specified width and height. Then, we use JavaScript to access the canvas and its context. With the context, we can set the fill style to red, and then use the fillRect() method to draw a rectangle on the canvas at the coordinates (10, 10) with a width of 180 and a height of 80.

This is just a basic example, you can use canvas to create more complex graphics, animations and games by using the various canvas API's such as paths, gradients, images, text, and more.
