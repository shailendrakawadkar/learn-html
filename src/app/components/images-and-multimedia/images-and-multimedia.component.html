Images and Multimedia

In this section, we will learn about the different HTML elements that are used to add images and multimedia to a webpage.

Images

The img tag is used to add images to a webpage.

Example:

<img src="image.jpg" alt="A beautiful landscape">

Image Gallery

The figure and figcaption tags can be used to create a image gallery.

Example:

<figure> <img src="image1.jpg" alt="Image 1"> <figcaption>Caption for Image 1</figcaption> </figure> <figure> <img src="image2.jpg" alt="Image 2"> <figcaption>Caption for Image 2</figcaption> </figure>

Videos

The video tag is used to add videos to a webpage.

Example:

<video src="video.mp4" controls></video>

Audio

The audio tag is used to add audio to a webpage.

Example:

<audio src="audio.mp3" controls></audio>
