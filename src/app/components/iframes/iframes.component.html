IFrames

An IFrame (Inline Frame) is an HTML document embedded inside another HTML document on a website. It allows you to embed other websites, documents, or media within your own webpage.

IFrame Example

In this example, we have used the <iframe> element to embed a Google Maps webpage within our own webpage.

          
  <iframe src="https://www.google.com/maps/embed?pb=!1m14!1m8!1m3!1d14031.464923693637!2d-96.8272068!3d32.7845472!3m2!1i1024!2i768!4f13.1!3m3!1m2!1s0x0%3A0x9f3a0c3e3b8e7d0a!2sDeep+Ellum!5e0!3m2!1sen!2sus!4v1535880019162" width="600" height="450" frameborder="0" style="border:0" allowfullscreen></iframe>

In this example, the src attribute is used to specify the URL of the webpage to be embedded within the <iframe> element. The width and height attributes are used to specify the size of the IFrame, and the frameborder attribute is used to specify whether or not the border should be displayed. Additionally, the allowfullscreen attribute is used to allow the embedded webpage to be viewed in fullscreen mode.
