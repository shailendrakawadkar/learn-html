HTML and Microdata

Microdata is a method of embedding structured data within HTML code to provide additional information about the content on a web page. Search engines, web browsers, and other software can use this data to understand the content and context of the page.

Using Microdata in HTML

Microdata is added to a web page using the 'itemscope', 'itemtype', and 'itemprop' attributes. The 'itemscope' attribute is used to define a new item, the 'itemtype' attribute is used to specify the item's type, and the 'itemprop' attribute is used to specify the properties of the item.

For example, to mark up a product on a webpage using microdata, we can use the following code:

          
  <div itemscope itemtype="http://schema.org/Product">
    <h1 itemprop="name">Product Name</h1>
    <img itemprop="image" src="product.jpg" alt="Product Image">
    <p itemprop="description">Product Description</p>
    <p><span itemprop="price">$50</span> <span itemprop="priceCurrency">USD</span></p>
  </div>

In this example, we have a 'div' element with 'itemscope' and 'itemtype' attributes that define a new item of type "Product". Inside the 'div', we have several elements with 'itemprop' attributes that specify the properties of the product such as 'name', 'image', 'description', 'price', and 'priceCurrency'.

By adding microdata to a web page, you can help search engines and other software understand the content and context of the page, which can improve the visibility of the page in search results and enable new features and functionality.
