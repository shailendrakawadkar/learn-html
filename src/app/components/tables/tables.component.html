HTML Tables

In this section, we will learn about the different HTML elements that are used to create tables on a webpage.

Table Elements

The table, tr, th, and td tags are used to create tables, table rows, table headers, and table cells.

Example:

          
  <table>
    <tr>
      <th>Header 1</th>
      <th>Header 2</th>
      <th>Header 3</th>
    </tr>
    <tr>
      <td>Row 1, Cell 1</td>
      <td>Row 1, Cell 2</td>
      <td>Row 1, Cell 3</td>
    </tr>
    <tr>
      <td>Row 2, Cell 1</td>
      <td>Row 2, Cell 2</td>
      <td>Row 2, Cell 3</td>
    </tr>
  </table>
