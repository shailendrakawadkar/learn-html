HTML and CSS

In this section, we will learn about the basics of HTML and CSS and how they work together to create a webpage.

HTML

HTML (Hypertext Markup Language) is the standard language used to create web pages. It provides the structure and content of a webpage. HTML elements are represented by tags, which are enclosed in angle brackets. Examples of HTML tags include 

<p>, <h1>, <div>, <a>, <img> etc.

HTML documents are made up of a series of elements, each of which is represented by an opening and closing tag. The content of the element is placed between these tags. For example, 

<p>This is a paragraph</p>
is an example of a paragraph element.

CSS

CSS (Cascading Style Sheets) is a language used to add style and layout to HTML documents. It allows for the separation of presentation from structure, which makes it easier to maintain and update a webpage. CSS styles are applied to HTML elements using selectors, which specify which elements the styles should be applied to.

CSS styles can be applied in a number of ways, including inline styles, internal stylesheets, and external stylesheets. Inline styles are added directly to an HTML element using the style attribute, while internal and external stylesheets provide a way to separate the styles from the HTML content.

Example:

          
  <html>
    <head>
      <link rel="stylesheet" type="text/css" href="styles.css">
    </head>
    <body>
      <header>
        <h1 class="page-title">My Website</h1>
      </header>
      <nav>
        <ul>
          <li><a href="#home">Home</a></li>
          <li><a href="#about">About</a></li>
          <li><a href="#services">Services&
          </a></li>
          </ul>
        </nav>
        <main>
          <section id="home">
            <h2>Home</h2>
            <p>Welcome to our website!</p>
          </section>
          <section id="about">
            <h2>About</h2>
            <p>Learn more about our company and our mission.</p>
          </section>
        </main>
      </body>
    </html>
            
          
            
    /* styles.css */
    
    .page-title {{'{'}}
      text-align: center;
      font-size: 36px;
      color: #333;
    {{'}'}}
    
    nav ul {{'{'}}
      list-style: none;
      display: flex;
      justify-content: space-between;
    {{'}'}}
    
    nav a {{'{'}}
      text-decoration: none;
      color: #333;
    {{'}'}}
    
    section {{'{'}}
      padding: 20px;
    {{'}'}}

As you can see in the example, the HTML provides the structure and content of the webpage, while the CSS is used to add style and layout. The 

<link>
tag in the HTML head is used to link to an external stylesheet, in this case "styles.css", which is where the styles for the webpage are defined. The class attribute is used to apply styles to specific elements, such as the "page-title" class applied to the header h1 element.
