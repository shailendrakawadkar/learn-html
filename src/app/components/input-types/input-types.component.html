Input Types

HTML forms are used to collect user input, and the <input> element is used to create various types of form controls. The type attribute is used to specify the type of input control.

Text Input

The most basic type of input is the text input, which is created using the <input type="text"> element. This type of input allows the user to enter a single line of text.

Password Input

A password input is created using the <input type="password"> element. This type of input is used to collect sensitive information, such as a password, and the text entered is obscured for security reasons.

Checkbox Input

A checkbox input is created using the <input type="checkbox"> element. This type of input allows the user to select multiple options from a group of options.

Radio Input

A radio input is created using the <input type="radio"> element. This type of input allows the user to select a single option from a group of options.

Select Input

A select input is created using the <select> element. This type of input allows the user to select one or more options from a list of options. The options within the select input are created using the <option> element.

Input Types Example

          
  <form>
    <label for="username">Username:</label>
    <input type="text" id="username" name="username">
    <br>
    <label for="password">Password:</label>
    <input type="password" id="password" name="password">
    <br>
    <label>Fruits:</label>
    <input type="checkbox" id="apple" name="fruit" value="apple">
    <label for="apple">Apple</label>
    <input type="checkbox" id="banana" name="fruit" value="banana">
    <label for="banana">Banana</label>
    <br>
    <label>Gender:</label>
    <input type="radio" id="male" name="gender" value="male">
    <label for="male">Male</label>
    <input type="radio" id="female" name="gender" value="female">
    <label for="female">Female</label>
    <br>
    <label for="country">Country:</label>
    <select id="country" name="country">
      <option value="usa">USA</option>
      <option value="canada">Canada</option>
      <option value="mexico">Mexico</option>
    </select>
    <br>
    <input type="submit" value="Submit">
  </form>

In this example, we have used various input types such as text, password, checkbox, radio, and select to create a simple form. By using the appropriate input type, we can ensure that the user input is collected in an organized and easy-to-use manner. Additionally, we have also used the label element to improve accessibility by providing a text description for each input.
