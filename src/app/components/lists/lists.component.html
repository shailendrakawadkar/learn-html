Lists

HTML lists are used to organize and structure content in a clear and easy-to-read format. They come in two types: ordered (numbered) and unordered (bulleted).

Ordered Lists

Ordered lists are created using the <ol> element, and each list item is represented by the <li> element. The default numbering style is decimal, but this can be changed using the type attribute.

Unordered Lists

Unordered lists are created using the <ul> element, and each list item is represented by the <li> element. The default bullet style is a disc, but this can be changed using the type attribute.

Lists Example

          
  <h2>My Favorite Fruits</h2>
  <ul>
    <li>Apples</li>
    <li>Bananas</li>
    <li>Oranges</li>
  </ul>
  
  <h2>Steps to Make a Sandwich</h2>
  <ol type="1">
    <li>Take two slices of bread</li>
    <li>Add your favorite fillings</li>
    <li>Close the sandwich and enjoy!</li>
  </ol>

In this example, we have used an unordered list to create a list of my favorite fruits, and an ordered list to create a list of steps to make a sandwich. Using the <ul> and <ol> elements and the <li> elements, we can easily create and structure lists of items in an easy-to-read format. Additionally, we can change the default numbering or bullet style by using the type attribute. Lists are a great way to organize and structure content, making it easy to understand and follow.
