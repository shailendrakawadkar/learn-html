Basic HTML Elements

In this section, we will learn about the basic HTML elements that are used to create the structure of a webpage.

Headings

HTML provides six levels of headings, which are denoted by the h1 to h6 tags.

Example:

<h1>Heading Level 1</h1>
<h2>Heading Level 2</h2>
<h3>Heading Level 3</h3>
<h4>Heading Level 4</h4>
<h5>Heading Level 5</h5>
<h6>Heading Level 6</h6>

Paragraphs

The p tag is used to create a paragraph of text.

Example:

<p>This is a paragraph of text.</p>
<p>This is another paragraph of text.</p>

Links

The a tag is used to create a link to another webpage.

Example:

<a href="https://www.example.com">Visit Example.com</a>

Images

The img tag is used to display an image on a webpage.

Example:

<img src="image.jpg" alt="A beautiful landscape">
