Text Formatting

In this section, we will learn about the different HTML elements that are used to format text on a webpage.

Bold Text

The b tag is used to make text bold.

Example:

<p>This is a paragraph of text. <b>This part is bold.</b></p>

Italic Text

The i tag is used to make text italic.

Example:

<p>This is a paragraph of text. <i>This part is italic.</i></p>

Underlined Text

The u tag is used to underline text.

Example:

<p>This is a paragraph of text. <u>This part is underlined.</u></p>

Strikethrough Text

The s tag is used to create a strikethrough effect on text.

Example:

<p>This is a paragraph of text. <s>This part has a strikethrough.</s></p>

Marked Text

The mark tag is used to highlight text.

Example:

<p>This is a paragraph of text. <mark>This part is highlighted.</mark></p>

Subscript Text

The sub tag is used to create subscript text.

Example:

<p>This is a paragraph of text. <sub>This part is subscript.</sub></p>

Superscript Text

The sup tag is used to create superscript text.

Example:

<p>This is a paragraph of text. <sup>This part is superscript.</sup></p>
